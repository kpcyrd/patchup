# patchup


## Design choices

- No code execution, neither an admin nor the hub service can execute code on the monitored devices
- Successor to Mozilla's mig project, but more limited and easier to setup and run
- Support for ephemeral nodes
- Works well with cloud-init
- Works well with ansible deployments
- Memory-safe code
- Reproducible pre-compiled binaries
